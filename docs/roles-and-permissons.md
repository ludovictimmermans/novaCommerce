NovaCommerce uses a clear, enterprise-style role system.

---

## USER

- browse products
- manage cart
- manage account
- place orders
- view order history
- download invoices

---

## MANAGER

- manage brands (add, edit, disable)
- manage categories (add, edit, disable)
- manage products (add, edit, disable)
- view stock
- increase stock
- correct stock

---

## ADMIN

Includes all MANAGER permissions plus:

- delete brand/category/product
- decrease stock
- manage users (view, disable, assign roles)
- refund orders
- view all orders

---

## SUPPORT

- view orders with issues
- update return status
